# 67. 二进制求和

## 题目描述

给你两个二进制字符串，返回它们的和（用二进制表示）。

输入为**非空**字符串且只包含数字 `1` 和 `0`。

**示例 1:**

> 输入: a = "11", b = "1"
> 输出: "100"

**示例 2:**

> 输入: a = "1010", b = "1011"
> 输出: "10101"

**提示：**

- 每个字符串仅由字符 `'0'` 或 `'1'` 组成。
- `1 <= a.length, b.length <= 10^4`
- 字符串如果不是 `"0"`，就都不含前导零。

---

## 一行解：int 和 bin

`int` 当显式指定第二个参数 `base` 时，第一个参数 `x` 必须是一个字符串。

`bin` 可以返回整数的二进制表示，以 `'0b'` 开头。

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        return bin(int(a, 2) + int(b, 2))[2:]
```
